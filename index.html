Connect Spotify

Tuneless uses the Spotify Web Playback SDK to stream full songs. You need a Spotify Premium account and a free Spotify Developer app.

  1. Go to developer.spotify.com/dashboard
  2. Create an app — any name, tick Web Playback SDK under APIs used
  3. In Settings → Redirect URIs, add exactly:
  4. Under User Management, add each listener's exact name + Spotify email — otherwise they get a 403 "User not approved"
  5. Save → copy your Client ID → paste below
